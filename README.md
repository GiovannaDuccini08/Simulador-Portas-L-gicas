# Simulador de Sistemas Binários e Portas Lógicas

Este projeto é um **simulador em Python** desenvolvido para auxiliar no aprendizado de **sistemas binários** e **portas lógicas fundamentais**, muito utilizadas em eletrônica digital, arquitetura de computadores e ciência da computação.

O programa permite ao usuário interagir via terminal, escolhendo portas lógicas e visualizando seus resultados, além de gerar **tabelas verdade** automaticamente.

---

## Conceitos abordados

- Sistemas binários (0 e 1)
- Álgebra Booleana
- Portas lógicas:
  - AND
  - OR
  - NOT
  - XOR
- Tabela verdade
- Estruturas básicas em Python
  - Funções
  - Laços
  - Condicionais
  - Validação de entrada

---

## Funcionalidades

- Simulação interativa das portas lógicas
- Validação de entradas (aceita apenas 0 ou 1)
- Geração de tabela verdade para cada porta
- Interface simples e intuitiva via terminal

---

## Menu do programa

1 - AND
2 - OR
3 - NOT
4 - XOR
5 - Tabela Verdade
